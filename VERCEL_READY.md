## 🚀 **Deploy to Vercel Now:**

### **Step 1: Install Vercel CLI**
```bash
npm i -g vercel
```

### **Step 2: Deploy**
```bash
cd quiz-frontend
vercel
```

### **Step 3: Follow Prompts**
- Link to existing project? **No**
- Project name: **quiz-app**
- Directory: **./** (current directory)
- Override settings? **No**

### **Step 4: Deploy to Production**
```bash
vercel --prod
```
